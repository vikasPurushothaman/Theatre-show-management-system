# BookMyShow SQL Assignment

## How to Run

```bash
# 1. Create schema
mysql -u root -p < schema.sql

# 2. Insert sample data
mysql -u root -p < sample_data.sql

# 3. Run P2 query
mysql -u root -p bookmyshow < queries_p2.sql
```

## Files
| File | Purpose |
|------|---------|
| `schema.sql` | All CREATE TABLE statements (P1) |
| `sample_data.sql` | Sample INSERT statements (P1) |
| `queries_p2.sql` | Show listing query (P2) |
